01.一隻馬
    馬象徵著追隨自己的熱情，朝著目標前進。你能知道自己真正要什麼，並不顧一切去實現它。除此之外，第一眼看到馬的人性慾也很強。	
01.一隻公雞
    第一眼看到公雞的人有冒險精神，也很有自信，很適合當領導者，除了激勵團隊士氣還能成為一個可靠支柱。
01.一隻螃蟹
    螃蟹暗示著一件事，我們無法總是走直線來達到我們的目的地，做事方法需要很多彈性和角度。如果你選到螃蟹，除了有上述特質外，也很敏感和害羞。
01.一隻螳螂	
    第一眼看到螳螂的人，很需要內心的平靜。現階段的你因為生活的混亂神經非常緊繃，渴望能呼吸到更多空氣。幸好你很懂得利用時間，知道怎麼讓你的生活回歸平衡。你同時也很聰明、有洞見，是不錯的決策者。
01.一隻狼	
    我們活在規律的生活裡，帶著得體的面具，但內心卻暗藏著自由的狂野。如果你第一眼看到狼，那你是很能平衡現實與夢想的人。不會過於天馬行空，也不會過於悲觀實際。你很容易跟別人打成一片，但同時競爭心也很強。
01.一隻狗狗	
    如果你第一眼看到狗，那你就如狗一般友善、忠誠、值得信任。但總是對別人好的你往往會忘了對自己好。你很有犧牲奉獻的精神，懂你的人會對於你的愛和關照心存感恩。
01.一隻鷹
    第一眼看到鷹的人很需要別人精神上的認可，也很愛好自由，需要內心的滋養，總是希望藉由各種方式讓自己內心更強大。	
01.一隻蝴蝶
    第一眼看到蝴蝶的人擁抱改變，雖然高敏感的你可能會因為環境的改變而受到心情上的影響。另外，你是個很喜歡逗樂身邊親朋好友的開心果。
01.一隻白鴿
    第一眼看到白鴿的人懂得何時前進，何時又該妥協。妥協對你來說不一定是認輸，而是讓事情更順利的方法。另外，有愛心的你很希望做點什麼來回饋社會。

02.馬
    【智慧型】對於自己有良好的控制能力，善於思考和推演，尤其在作任何判斷時，都會縝密的思考，公平且完全的客觀，是穩重而可以倚賴的人。  

    【外在】非常重視事情合理性，深信要解決問題必須找出問題的根源，快速地對症下藥，效率十足。   

    【內在】對於生活周遭的事物感覺敏銳，喜歡榮耀、被稱讚的感覺，對於自己拿手的事物，有時會有些自我中心。只會被心裡認同的人給說服。    
02.牛
    【藝術型】思考模式旁人有時難以理解，思維常常沒有邏輯可言，很難抓透在想什麼，天馬行空的幻想對藝術型來說是家常便飯，對美的事物直覺很強。

    【外在】感覺至上，很重視直覺的人。重視第一印象，第一眼的感覺會影響他們對人的觀感。處理很多事情以直覺為準則。

    【內在】對於生命充滿著熱情和純粹的天真，並有百分之百的理想性。相信世界是美好的，每件新事物都是嶄新的契機，對現實社會的黑暗面感到不可思議，甚至不能接受。
    
02.熊
    【自由型】依靠堅毅不拔取得成功，富創造力，希望做需要做和想要做的事情。冷靜地堅強、責任心強、關心他人。因其堅定的原則而受尊重。

    【外在】對人應對流暢，脾氣好，能商量事情，不過原則是不妨礙到自己私人時間。朋友不多，但是幾乎個個都是知己。

    【內在】常常表面隱藏，內心卻在忍受別人所看不到之壓力，渴望有任何人事物都能有一個「自己的」，喜歡幻想。

03.鴿子
    鴿子象徵著和平與希望，代表你有著純潔的靈魂，總是關注著美好的事物、看到每個人的優點，比起物質上的生活，你更在乎的是如何鼓勵與激發人他人朝向充滿希望的理想人生
03.蝴蝶
    蝴蝶代表著美麗與快樂，你有著讓人著迷的美麗靈魂，你知道生命是短暫的，於是你努力珍惜當下，你討厭無聊的生活，所以你會充分利用時間來充實自己、追求不平凡的每一天
03.老鷹
    老鷹代表著自由與無畏風險的精神，你獨立堅強的個性常常吸引著身邊的人，你就像是個天生的領袖擁有著高貴優雅的靈魂，求知慾很高且不喜歡被約束，勇於挑戰的你會竭盡所能去達到自己的目標。
03.狗狗
    狗狗代表著忠誠與家庭，對於你所愛的人非常無私，你把他們的快樂與幸福看得比自己的人生還重要，你的個性活潑又有愛心，你總是散發著開朗與喜樂的光芒，周圍的人跟你相處感到安心可靠。
03.狼
    狼代表著孤獨與神秘，你很少會向他人吐露自己真實的情緒和感受，你很珍惜自己的時間和隱私，交朋友時你偏向於寧缺勿濫，寧願保持且珍惜著小小的社交圈，你充滿智慧且個性沉穩冷靜，內心溫柔的你其實很需要被保護
03.螳螂
    螳螂是許多文化中的神聖動物，如果你選的是螳螂，代表你是一個很享受孤獨的人，你喜歡獨自沉浸在學習的世界裡不想被打擾，也不想花太多時間在社交上，但不代表你不擅長與人互動，你只是覺得跟人閒聊打屁很無趣，而且你很聰明，也有計算、掌管事情的能力
